Lighting the braziers at the Ice Demon in Chambers of Xeric is a team effort, but nobody
can actually see how much kindling has gone in - so it turns into guesswork and calling
out numbers. Team Kindling Tracker keeps a running total above each brazier, adds up
everyone's contributions across the party, and shows who's still carrying kindling.

## Features

| | |
|---|---|
| 🔥 **Live brazier totals** | The party-wide amount of kindling fed into each of the four braziers, drawn right above it. |
| 🤝 **Whole-party counts** | Everyone's contributions are shared over the RuneLite Party API, so every client shows the same numbers. |
| 🎒 **Who's holding kindling** | Optional overlay box listing how much kindling each teammate still has in their inventory. |
| 🧍 **Above-head counts** | Optional kindling-in-hand number over each teammate, with adjustable height, size and bold. |
| ♻️ **Hands-off** | Resets itself at each Ice Demon and clears the inventory displays once the demon thaws or dies. |

> For the brazier totals to be complete, everyone feeding needs the plugin and the same
> RuneLite party - your client can't see other players' inventories any other way.
